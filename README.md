# 🏺 Juthour Cloud POS | نظام جذور السحابي
![Status](https://img.shields.io/badge/Status-Live-success)
![Platform](https://img.shields.io/badge/Platform-Arise_Cloud-blue)

نظام مبيعات متطور (Point of Sale) مصمم خصيصاً لـ **محل جذور (Juthour)** لإدارة عمليات بيع العطور والبخور والهدايا بكفاءة عالية.

## 🚀 المميزات التقنية
- **الربط السحابي (Cloud Identity):** النظام مربوط بـ **Firebase Firestore** لتسجيل المبيعات لحظياً (Real-time).
- **تصميم فاخر (Premium UI):** واجهة مستخدم داكنة بلمسات ذهبية تليق ببراند العطور الراقية.
- **التوافق التام:** يعمل بسلاسة على المتصفح ومن خلال الهواتف الذكية.
- **الأمن الرقمي:** تشفير البيانات وحمايتها عبر سيرفرات جوجل العالمية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).
- **Backend:** Firebase (Firestore Database).
- **Deployment:** GitHub Pages.

## 📱 طريقة الاستخدام
1. افتح رابط النظام على المتصفح.
2. اختر المنتج المطلوب (عطر، بخور، هدايا).
3. سيتم تسجيل العملية فوراً في قاعدة البيانات السحابية مع السعر والوقت.
